LiBot Customization Module ==========

A module providing commands and features allowing per-guild customization of other LiBot features directly from LiBot.

Module data:
	Providers --------------------------
	- [depends on] CustomizationsProvider
	- AutoRoleProvider
	- GreeterProvider
	
	Listeners --------------------------
	- AutoRoleListener
	- GreeterListener
	
	Commands ---------------------------
	- AutoRoleCommand
	- DisableCommand
	- DjRoleCommand
	- EnableCommand
	- GreeterCommand
	- SetPrefixCommand
