LiBot Administrative Module ==========

A module providing commands and features allowing administration of LiBot directly from LiBot. Most of these require your user ID to be listed in BOT_SYSADMINS. They are also not listed in the help page for users who are not listed as sysadmins.

Module data:
	Providers --------------------------
	- [depends on] ConfigurationProvider
	
	Listeners --------------------------
	(none)
	
	Commands ---------------------------
	- EvaluateCommand
	- GlobalDisableCommand
	- GlobalEnableCommand
	- KillProcessCommand
	- ShutdownCommand
